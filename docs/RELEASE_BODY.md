## Local Music Hub 0.13.21

### Volume slider (Spotify-like)
- **Instant** volume changes with **no buzz, static, or ticks**.
- Master volume is applied in the audio engine with a tiny (~1.5 ms) dezipper — not an audible fade.
- Volume bar fills **edge-to-edge at 100%**.

### Also since 0.13.17
- WAID-style **Download and run installer** updates (full quit, not tray).
- FLAC / OGG playback and tag fixes for Windows Details.
- Playlist play reliability, folder watch, album editor, and CLI `--playlist` / `--volume` support.

Install **`LocalMusicHub-Setup-0.13.21.exe`** from this release. Your library under `%LocalAppData%\LocalMusicHub\` is kept.
